개인정보처리방침

초간편 가계부 · 최종 수정일: 2025년 7월

핵심 요약

이 앱은 모든 데이터를 사용자의 기기 내에만 저장합니다. 외부 서버로 개인정보를 전송하거나 수집하지 않습니다.

1. 수집하는 정보

초간편 가계부는 어떠한 개인정보도 수집하지 않습니다. 앱에 입력된 지출·수입·예산 데이터는 전적으로 사용자의 기기(로컬 저장소)에만 저장되며, 개발자 또는 제3자에게 전송되지 않습니다.

2. 데이터 저장 방식

  • 모든 거래 내역, 예산, 카테고리 정보는 기기 내 로컬 저장소(앱 내부 저장소)에 저장됩니다.
  • 앱을 삭제하면 저장된 모든 데이터가 함께 삭제됩니다.
  • 클라우드 동기화 또는 외부 서버 전송 기능은 존재하지 않습니다.

3. 제3자 제공

수집된 개인정보가 없으므로, 어떠한 제3자에게도 정보를 제공하거나 공유하지 않습니다.

4. 인터넷 권한 사용 목적

앱은 INTERNET 권한을 선언하고 있으나, 이는 Capacitor 프레임워크의 기본 요구 사항이며 실제 외부 네트워크 통신에 사용되지 않습니다.

5. 아동 개인정보 보호

이 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 방침 변경 안내

본 방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 안내드립니다.
